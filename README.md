# Simulador de Caixa Eletrônico (Portugol)

Desenvolvimento de um sistema bancário estruturado para praticar regras de negócio e segurança lógica.

## Funcionalidades Implementadas
* **Autenticação:** Proteção de operações por senha numérica (`3589`).
* **Validação de Saldo:** Travas de segurança para impedir saques ou transferências maiores que o saldo.
* **Operações Bancárias:** Depósito, Saque, Transferência e Extrato.
* **Experiência do Usuário:** Saudações personalizadas e mensagens de erro intuitivas.

## Conceitos Técnicos
* **Modularização:** Organização do código em funções (`inicio`, `sacar`, `depositar`, etc).
* **Validação de Dados:** Filtros para garantir que valores informados sejam sempre positivos.
* **Recursividade e Fluxo:** Chamada de funções para repetir processos em caso de erro de senha.

## Tecnologias
* **Linguagem:** Portugol
* **Ambiente:** Portugol Studio
